Introduction

You can view and change your Joulescope UI settings in the Preferences Dialog. Preferences allow you to change how the Joulescope UI operates, including instrument behavior, colors, and fonts. Profiles group Preference settings so that you can quickly switch between modes of operation. The Joulescope UI ships with three standard Profiles:

  • defaults: The default settings used by all Profiles, unless a Profile overrides the default setting.
  • Multimeter: The settings for the "Multimeter" view.
  • Oscilloscope: The settings for the "Oscilloscope" view.

Usage

Select the Profile that you want to view or edit using the drop-down list. Select the Preferences Group on the left-hand side. View or change the setting on the right-hand side. Changes only apply to the selected Profile.

Select the active Profile using the "Activate" button. You can also select the active Profile under the View menu.

You can add custom Profiles using the "New" button. The View menu will also display custom Profiles.
